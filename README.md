# gateway-dns-operator

Kubernetes operator that syncs Gateway API HTTPRoutes to AdGuard Home DNS rewrites for split-horizon DNS.

## Description

This operator walks over all HTTP Routes, obtains their attached gateway's Cilium VIP address, and syncs it into AdGuard Home via REST API.
